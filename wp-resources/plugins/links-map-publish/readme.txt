Auttomatically update the link_updated field when adding or editing a link, so you can use Links as a linklist.
Also provides a form for adding the map coordinates

== Description ==

The plugin updates the link_updated field when adding or editing a link. In this way you can use Wordpress Links as a linklist, sync to your articles.
