Multilingual Dictionary Theme

The Webonary project intends to give language groups the ability to put their bilingual or multilingual dictionaries on the web with a minimum of technical help. Multilingual-Dictionary is a Webonary theme, and is based on the theme Star-Brite.

Theme Installation

Upload the theme files to your wordpress theme folder wp-content/themes and activate the theme in your wordpress admin panel.

Plugins Installation

You will need the WordPress plugin SIL Dictionary Infrastructure.

This theme also requires get-a-post.

Assuming you have a bilingual or multilingual dictionary, you will also need to download, install, and activate qTranslate.

The Yi site also makes use of WP Page Numbers.
